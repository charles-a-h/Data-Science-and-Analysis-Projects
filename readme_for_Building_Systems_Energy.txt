This readme file accompanies the Building Systems Energy data science project.
All of the data files are included to perform the analysis for this project with the exception of one, due to a large file size.

Please refer to the following website url to download the data:
https://www.kaggle.com/c/ashrae-energy-prediction/data

After you have downloaded this data, please rename the training data to:
"06-ASHRAE_Great_Energy_Predictor_3_train"

This file is approximately 663Mb in size.
